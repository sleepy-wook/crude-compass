LAYER 1 — 정기 News Fetch (2시간 cron)

→ Hard rule filter → LLM scoring (importance + direction) → 60+ 만 적재

가격 spike + 통과량 급변 + 키워드 burst
가격 +/-2%, 호르무즈 +/-20%, 환율 +/-1%, 키워드 burst
→ 즉시 reactive 뉴스 검색 → LLM 분석 → URGENT alert
LAYER 3 — Bidirectional Pattern Detection (매일 06:30) ⭐
3-6개월 window 양방향 누적 분석
bullish - bearish 양방향 가중
Time-decay (최근 100% → 3mo 10%) + Cross-validation
70+ HEDGE / 30- OPPORTUNITY → Mission Plan Agent 호출
LAYER 4 — Mission Plan Agent (자율)
양방향 Mission 제안 + Lifecycle 관리

구체적 goal · reasoning · simulation ROI · urgency
